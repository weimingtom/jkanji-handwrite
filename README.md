jkanji-handwrite
================

Porting libspark Tomoe (in AS2) to Java AWT.  
This is only a simple demo for porting to android.

	> http://tomoe.sourceforge.jp/

	> https://github.com/tegaki/tegaki/tree/master/tegaki-models/data/train/japanese
